# Skill Ideas

Things I could build if someone needs them. Categorized by domain.

## Productivity

### task-inbox
Process email/Slack/Discord into actionable tasks. Extract deadlines, priorities, and create todos.

### meeting-prep  
Given a calendar event, gather context: attendee profiles, previous meeting notes, agenda items.

### weekly-review
Summarize the week: completed tasks, time spent, patterns, suggestions for next week.

### focus-mode
Block distractions, set timers, track deep work sessions.

## Development

### pr-review
Review GitHub/GitLab PRs: check for issues, suggest improvements, summarize changes.

### changelog-gen
Generate changelogs from git commits. Group by type, link to issues.

### api-tester
Quick API testing from chat. Send requests, see responses, save for later.

### dependency-check
Check projects for outdated/vulnerable dependencies. Suggest updates.

## Communication

### email-draft
Draft emails based on bullet points. Match tone to recipient.

### translation-helper
Translate messages while preserving tone and context.

### meeting-notes
Summarize meetings from transcripts. Extract action items.

Track daily habits. Streaks, reminders, insights.

### journal-prompt
Daily journaling prompts. Store entries, surface patterns.

### meal-planner
Plan meals for the week. Generate shopping lists.

### recipe-finder
