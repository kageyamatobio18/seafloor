# OpenClaw Quickstart Generator

A simple web tool that generates OpenClaw configuration files based on user selections.

## Features

- Choose your messaging channel (Telegram, WhatsApp, Discord, Signal)
- Choose your AI model provider (Anthropic, OpenAI, OpenRouter)
- Paste your tokens/API keys
- Get a ready-to-use config file
- Copy to clipboard with one click
- Clear next steps

## Deploy

This is a single HTML file with no dependencies. Deploy anywhere:

```bash
# Vercel
vercel

# Netlify
netlify deploy

# GitHub Pages
# Just push to a gh-pages branch

# Any static host
# Just upload index.html
```

## Local Preview

```bash
npx http-server . -p 8080
# or
python -m http.server 8080
```
